# CSS Transforms, Transitions, and Animations

## CSS transform
It allow us to translate, rotate, scalie and skew and effect in element size, shap and position.

CSS 2D transforms methods:

* translate()
* translateX()
* translateY()
* rotate()
* scale()
* scaleX()
* scaleY()
* skew()
*  skewX()
*   skewY()
* matrix()
